## Descripción del Proyecto

Software de simulación para sistemas de control de trenes de metro. La lógica de control se desarrolla usando Python, el frontend usa React + TypeScript, y la plataforma de gestión y análisis usa SpringBoot + Vue. Los tres subsistemas se comunican a través de colas de mensajes MQTT y admiten dispositivos físicos externos como consolas de control. Admite operaciones de simulación para vías, desvíos, señales, plataformas, etc. Puede crear trenes, definir varios tipos de eventos anormales, admite inicio de sesión multi-usuario, admite edición y carga de exámenes, admite vincular operaciones de consola para controlar trenes, y todas las operaciones se registran y analizan automáticamente para generar informes.
