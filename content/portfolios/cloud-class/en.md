## Project Description

"Qihang Education Cloud Classroom" is a learning system based on mobile devices that supports offline learning and instant learning progress recording (supports iOS, Android and iPad). Users enter the application through their account, select courses of interest, and can easily start learning.

## Key Features

- Supports offline viewing. After course download is complete, courses can be viewed without network connection.
- Supports download while playing. When network conditions are good, courses can be viewed directly while being downloaded synchronously.
- Unified learning records. Learning records from mobile devices and browsers are completely unified.
