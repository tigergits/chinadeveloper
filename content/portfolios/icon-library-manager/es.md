## Descripción del Proyecto

ABBIcon Pro incluye ABB Icon Library Manager, ABB Image Icon Converter, ABB Icon Explorer, ABB Web Icon Extractor.

Comparación de características de las familias de productos ABB.

Gestionar bibliotecas de iconos como el Explorador de Windows
- Construir y mantener bibliotecas de iconos según se desee, sin límite en el número de iconos en cada biblioteca. 
- Gestionar bibliotecas de iconos, categorías de bibliotecas con una estructura jerárquica como el Explorador de Windows. 
- Buscar bibliotecas de iconos en el disco duro(s) de la PC local. 
- Escanear y eliminar iconos duplicados. 
- Modificar iconos en ABBIcon Pro con aplicación de edición de iconos. 
- Buscar iconos por palabras clave, tamaño y parámetros de color. 

Crear iconos desde archivos en 30 formatos

Hay 5 formas de extraer o crear iconos usando ABBIcon Pro: 
- (1) Extraer iconos de archivos en formato EXE, DLL, OCX, CPL, ICL, ICO, ANI, CUR. 
- (2) Crear iconos desde imágenes en 30 formatos. (haga clic aquí para ver detalles). 
- (3) Extraer iconos de páginas web, solo haga clic con el mouse, todas las imágenes en la página web se convertirán en iconos. 
- (4) Crear iconos desde captura de pantalla, como pantalla completa, ventanas, objeto, área. 
- (5) Extraer iconos de directorios o todo el disco duro(s). 

(*) Los iconos se agregan fácilmente a la ventana de ABBIcon Pro arrastrando archivos de soporte desde el Explorador de Windows o cualquier carpeta abierta. 

Exportar iconos a archivos en 17 formatos

Exportar iconos a otras bibliotecas de iconos. Exportar iconos a imágenes en 17 formatos. (haga clic aquí para ver detalles) 

(*) Exportar iconos arrastrando iconos al Explorador de Windows o cualquier carpeta abierta.
