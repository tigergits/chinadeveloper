## プロジェクト概要

ABBIcon Pro には、ABB Icon Library Manager、ABB Image Icon Converter、ABB Icon Explorer、ABB Web Icon Extractor が含まれています。

ABB 製品ファミリーの機能比較。

Windows エクスプローラーと同じようにアイコンライブラリを管理
- 必要に応じてアイコンライブラリを構築・維持し、各ライブラリのアイコン数に制限はありません。 
- Windows エクスプローラーと同じように、階層構造でアイコンライブラリ、ライブラリカテゴリを管理します。 
- ローカル PC のハードディスクでアイコンライブラリを検索します。 
- 重複アイコンをスキャンして削除します。 
- ABBIcon Pro でアイコン編集アプリケーションを使用してアイコンを変更します。 
- キーワード、サイズ、色のパラメータでアイコンを検索します。 

30 形式のファイルからアイコンを作成

ABBIcon Pro を使用してアイコンを抽出または作成する方法は 5 つあります： 
- (1) EXE、DLL、OCX、CPL、ICL、ICO、ANI、CUR 形式のファイルからアイコンを抽出します。 
- (2) 30 形式の画像からアイコンを作成します。（詳細はこちらをクリック）。 
- (3) ウェブページからアイコンを抽出します。マウスをクリックするだけで、ウェブページ内のすべての画像がアイコンに変換されます。 
- (4) 全画面、ウィンドウ、オブジェクト、領域などの画面キャプチャからアイコンを作成します。 
- (5) ディレクトリまたはハードディスク全体からアイコンを抽出します。 

(*) Windows エクスプローラーまたは任意の開いているフォルダからサポートファイルをドラッグすることで、アイコンを簡単に ABBIcon Pro のウィンドウに追加できます。 

17 形式のファイルにアイコンをエクスポート

アイコンを別のアイコンライブラリにエクスポートします。アイコンを 17 形式の画像にエクスポートします。（詳細はこちらをクリック） 

(*) アイコンを Windows エクスプローラーまたは任意の開いているフォルダにドラッグしてアイコンをエクスポートします。
