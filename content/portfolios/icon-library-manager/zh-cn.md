## 项目描述

ABBIcon Pro 包括 ABB 图标库管理器、ABB 图像图标转换器、ABB 图标浏览器、ABB 网页图标提取器。

ABB 产品系列功能对比。

像 Windows 资源管理器一样管理图标库
- 根据需要构建和维护图标库，每个库中的图标数量没有限制。 
- 像 Windows 资源管理器一样，使用分层结构管理图标库、库类别。 
- 在本地 PC 的硬盘上搜索图标库。 
- 扫描并消除重复图标。 
- 在 ABBIcon Pro 中使用图标编辑应用程序修改图标。 
- 通过关键字、大小和颜色参数搜索图标。 

从 30 种格式的文件创建图标

使用 ABBIcon Pro 有 5 种提取或创建图标的方法： 
- (1) 从 EXE、DLL、OCX、CPL、ICL、ICO、ANI、CUR 格式文件中提取图标。 
- (2) 从 30 种格式的图像创建图标。（点击此处查看详情）。 
- (3) 从网页中提取图标，只需点击鼠标，网页中的所有图像都将转换为图标。 
- (4) 从屏幕捕获创建图标，例如全屏、窗口、对象、区域。 
- (5) 从目录或整个硬盘提取图标。 

(*) 通过从 Windows 资源管理器或任何打开的文件夹拖放支持文件，可以轻松将图标添加到 ABBIcon Pro 的窗口中。 

导出图标为 17 种格式文件

将图标导出到另一个图标库。将图标导出为 17 种格式的图像。（点击此处查看详情） 

(*) 通过将图标拖放到 Windows 资源管理器或任何打开的文件夹来导出图标。
