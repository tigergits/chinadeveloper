## Project Description

Icon Office is a professional and powerful icon conversion and management tool, With Icon Office, you can: 

Manage icon libraries

- Build and maintain icon libraries as desired with no limit on the number of icons in each library. 
- Manage icon libraries, library categories with a hierarchical structure like Windows Explorer. 
- Search icon libraries in hard disk(s) on local PC. 
- Scan for and eliminate duplicate icons. 
- Search icons by keywords. 

Extract icons

There're 5 ways to extract or create icons using Icon Office: 

- Extract icons from EXE, DLL, ICL, ICO, CUR format files. 
- Create icons from BMP, GIF, JPG, PNG format files. 
- Extract icons from directories or whole hard disk(s). 

Export icons

- Export icons to another icon libraries 
- Export icons to ICO, ICL, BMP, JPG, GIF, PNG format files
