## Descripción del Proyecto

[OBSOLETO]

PAD (Portable Application Description) era un formato XML estándar ampliamente utilizado hace muchos años para definir información de software shareware y freeware. Los autores de software podían crear un archivo PAD para su aplicación según este estándar, y luego enviarlo a numerosos sitios web de descarga de software que admitían el formato PAD. Sin embargo, el shareware como modelo de distribución se ha vuelto en gran medida obsoleto. Este software fue desarrollado en 2005 usando Delphi 7 (Object Pascal) para ayudar a los desarrolladores de software a enviar automáticamente su software a más de 200 sitios de este tipo.

¿Qué es un archivo PAD?

Un archivo PAD (Portable Application Description) es un archivo XML estandarizado que contiene información estructurada sobre una aplicación de software, actuando como una "tarjeta de presentación" digital para que los desarrolladores enumeren detalles como nombre, versión, descripción y URLs de descarga, que los sitios y directorios de descarga leen y usan automáticamente para crear listados, ahorrando tiempo y asegurando consistencia. Es un formato abierto de la Association of Shareware Professionals (ASP) utilizado para automatizar el envío y actualización de software en muchas plataformas.

PADSubmitter automatiza completamente el proceso de envío de archivos PAD de software a más de 100 sitios populares de shareware y freeware. Simplemente ingrese una URL que apunte a su archivo PAD y una selección de categoría. El proceso de promoción del software es completamente automático.

Para un desarrollador de shareware, es bien sabido que la creación de un nuevo software a menudo toma mucho menos tiempo que su promoción. También se sabe que una gran parte del tiempo dedicado a la promoción no siempre se gasta de manera muy efectiva. Pero hay cientos de sitios de shareware/freeware disponibles y nuevos aparecen cada semana. Generalmente se dedica mucho tiempo a llenar formularios rutinarios y mecánicos al enviar la información de su software a numerosos sitios y archivos de software.

## Características Principales

- Gestionar archivos pad de múltiples softwares
- Envío automático a más de 200+ sitios (ver lista de sitios web compatibles)
- Facilidad de uso
- Un informe detallado del trabajo realizado
- Envío de un número ilimitado de URLs de PAD
