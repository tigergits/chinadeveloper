## プロジェクト概要

[非推奨]

PAD（Portable Application Description）は、何年も前にシェアウェアやフリーウェアの情報を定義するために広く使用されていた標準 XML 形式でした。ソフトウェアの作者は、この標準に従ってアプリケーションの PAD ファイルを作成し、PAD 形式をサポートする多数のソフトウェアダウンロード Web サイトに送信できました。しかし、シェアウェアは配布モデルとしてほぼ時代遅れになっています。このソフトウェアは 2005 年に Delphi 7（Object Pascal）を使用して開発され、ソフトウェア開発者がソフトウェアを 200 以上のそのようなサイトに自動的に送信するのを支援することを目的としています。

PAD ファイルとは？

PAD ファイル（Portable Application Description）は、ソフトウェアアプリケーションに関する構造化された情報を含む標準化された XML ファイルで、開発者のデジタル「名刺」として機能し、名前、バージョン、説明、ダウンロード URL などの詳細をリストします。ダウンロードサイトやディレクトリは自動的に読み取り、これらの情報を使用してリストを作成し、時間を節約し、一貫性を確保します。これは、Association of Shareware Professionals（ASP）からのオープン形式で、多くのプラットフォームでソフトウェアの送信と更新を自動化するために使用されます。

PADSubmitter は、ソフトウェア PAD ファイルを 100 以上の人気のあるシェアウェアおよびフリーウェアサイトに送信するプロセスを完全に自動化します。PAD ファイルを指す URL とカテゴリの選択を入力するだけです。ソフトウェアのプロモーションプロセスは完全に自動化されています。

シェアウェア開発者にとって、新しいソフトウェアの作成は、そのプロモーションよりもはるかに少ない時間で済むことがよく知られています。また、プロモーションに費やされる時間の大部分が常に非常に効果的に使われているわけではないことも知られています。しかし、数百のシェアウェア/フリーウェアサイトが利用可能で、毎週新しいサイトが出現しています。ソフトウェア情報を多数のソフトウェアサイトやアーカイブに送信する際、通常、ルーチンで機械的なフォーム入力に多くの時間が費やされます。

## 主な機能

- 複数のソフトウェアの pad ファイルを管理
- 200+ サイトへの自動送信（サポートされている Web サイトのリストを参照）
- 使いやすさ
- 完了した作業の詳細レポート
- 無制限の数の PAD URL の送信
