## Project Description

A drawing board program developed with QT/C++ (supports drawing and painting directly on Office documents, images, and videos).

## Key Features

- The drawing board can be divided into many pages, each page can contain many objects, objects support: images, brushes (single stroke, continuous stroke), videos, OFFICE objects
- Supports directly printing Office documents (WORD, EXCEL, PPT) to generate multi-page drawing board files
- Can freely draw with a pencil directly on the drawing board
- Videos and OFFICE objects can be directly opened for viewing or playback, objects can be moved, scaled, rotated, and drawn within objects
- Supports generating a folder into a standalone file on a USB drive, this file can only be opened by the drawing board program and can be directly embedded as an object in the drawing board
