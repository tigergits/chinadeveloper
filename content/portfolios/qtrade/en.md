## Project Description

A system developed with QT for graphical analysis and prediction based on stock market data.

## Key Features

- Automatically download and update stock market data
- Various charts (candlestick charts, various stock data charts, analysis charts)
- Various original stock data analysis charts, charts can be animated, visually displaying stock trends
- Some charts support 3D
- Powerful customization features
- Based on various data models, predict and backtest stock price trends based on historical data
