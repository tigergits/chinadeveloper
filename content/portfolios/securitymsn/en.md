## Project Description

Secure MSN encrypted with OpenSSL, directly using the MSN protocol.

## Key Features

- Secure MSN encrypted with OpenSSL, directly using the MSN protocol
- Supports Windows, Linux, macOS
- MSN login
- One-on-one chat, multiple chats (one-to-one RSA secure encryption)
- Encrypted file transfer
- Developed using QT/C++
